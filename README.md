# Flashcard API

## Introduction
The Flashcard API is designed to help users implement spaced repetition for effective studying and memorization. It offers various endpoints to manage flashcards, decks, and study progress, applying the spaced repetition algorithm for optimized learning.

## Base URL
The base URL for the Flashcard API is `http://localhost:5000/api/v1`.

## Authentication
The API requires authenticaion to access its resources. Users need to obtain an access token by logging in through the authentication endpoint. The token should be included in the `Authorization` header of each request.
